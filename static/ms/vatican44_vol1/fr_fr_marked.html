Suppressions et ajouts
Suppressions

Le scribe du manuscrit de Vatican indique les suppressions de plusieurs façons : une ligne horizontale barrant le mot ; une ligne horizontale au-dessus du mot ; des points au-dessus du mot ; des traits en diagonale, etc.

Il corrige le plus souvent ses erreurs en réécrivant sur le mot erroné :

- Ici, le scribe a d’abord écrit אומר - qui est le mot suivant dans le texte - avant de transformer ce mot en תחלה en écrivant par-dessus.

- Ici, le scribe a effacé le mot “את” qui n’apparaît pas dans le texte, par deux lignes diagonales au-dessus du mot.

Pour indiquer la suppression d’une lettre ou d’un mot dans le manuscrit, sélectionnez le texte supprimé avec la souris, puis cliquez sur le bouton « supprimé » (-) en-dessous de la boîte de texte. Vous pouvez également cliquer sur le bouton (-) puis taper le texte supprimé entre les parenthèses.

- Ici, la suppression doit être signalée de la façon suivante : כרכי (הים) סדום

- Le scribe a écrit א avant de le corriger en ע. Un tel exemple doit être marqué ה(א)עליונים.

Texte supprimé illisible

Si la lettre ou le mot ne peuvent être identifiés, appuyez une fois sur le bouton « Supprimé », afin d’indiquer que quelque chose a été supprimé à cet endroit, puis continuez.

Ajouts

Les ajouts peuvent être interlinéaires (en-dessus ou en-dessous de la ligne) ou marginales (dans les marges).

Le scribe du manuscrit de Vatican insère le plus souvent les ajouts les plus courts au-dessus de la ligne, tandis que les passages plus longs sont ajoutés dans les marges.

- Ici, le scribe a inséré le mot לו qui avait été oublié pendant la copie.

- Ici, le mot צדקו a été ajouté dans la marge, et sa position dans la ligne est signalée par une flèche au-dessus la ligne, après למען.

Afin de signaler un ajout, utilisez le bouton [+] en-dessous de la ligne de texte, et écrivez le mot inséré entre les crochets. Si vous devez signaler qu’une lettre ou un mot transcrits doivent être considérés comme un ajout, sélectionnez-les avec la souris, puis cliquez sur le bouton [+].

- l’addition doit être marquée: אתה [רוצה] להודיע

Texte endommagé ou illisible

Les manuscrits ont traversé les siècles, et n’en sortent pas souvent indemnes. Ils sont parfois déchirés, et l’encre peut s’être effacée ou estompée.

Pour marquer des lettres non-identifiables (déchirées ou estompées), cliquez sur le bouton et insérez des espaces entre les chevrons, selon votre estimation de la longueur du texte illisible. Par exemple:

- אם יש ל <>

Un doute concernant la transcription

‘douteux’

Le bouton du doute est destiné aux cas où vous n’êtes pas certain si la transcription automatique a bien identifié une lettre dans le manuscrit. Essayez d’utiliser ce bouton le plus rarement possible, et seulement dans des cas particulièrement difficiles.

Pour indiquer le doute, utilisez la souris pour sélectionner la lettre puis cliquez sur {douteux} {?}. Vous pouvez également cliquer sur le bouton et taper la lettre la plus probable entre les accolades.

- אם יש ל{עם}
